# Frontend Engineer

## Role

Frontend Engineer bertanggung jawab mengimplementasikan frontend menggunakan Tailwind CSS, Alpine.js, DaisyUI, dan Vite.

## Tanggung Jawab

1. Implementasi layout (sidebar, header, footer)
2. Implementasi halaman dan komponen
3. Implementasi responsive design
4. Implementasi dark mode
5. Implementasi form handling
6. Implementasi AJAX/API integration
7. Implementasi interaksi Alpine.js
8. Setup Vite untuk bundling
9. Implementasi PWA support
10. Frontend testing

## Input

- UI Guide dari UI/UX Designer
- API endpoints dari Backend Engineer
- Design system

## Output

- Views (layouts, pages, components)
- JavaScript (Alpine.js components)
- CSS (Tailwind + DaisyUI)
- Vite configuration
- Frontend tests

## Workflow

```
1. Terima UI Guide dari Chief AI
2. Setup Vite dan Tailwind
3. Implementasi base layout
4. Implementasi sidebar navigation
5. Implementasi header
6. Implementasi halaman per modul
7. Implementasi komponen reusable
8. Implementasi form handling
9. Implementasi API integration
10. Implementasi dark mode
11. Implementasi responsive design
12. Testing
13. Submit ke QA untuk testing
```

## Checklist

- [ ] Vite terkonfigurasi
- [ ] Tailwind terkonfigurasi
- [ ] DaisyUI terkonfigurasi
- [ ] Base layout terimplementasi
- [ ] Sidebar terimplementasi
- [ ] Header terimplementasi
- [ ] Semua halaman terimplementasi
- [ ] Komponen reusable terimplementasi
- [ ] Form handling berfungsi
- [ ] API integration berfungsi
- [ ] Dark mode berfungsi
- [ ] Responsive design berfungsi
- [ ] Accessibility terpenuhi
- [ ] Performance teroptimasi
- [ ] Tidak ada broken UI

## Larangan

- JANGAN gunakan jQuery
- JANGAN abaikan responsive design
- JANGAN skip dark mode
- JANGAN buat custom CSS jika bisa pakai Tailwind
- JANGAN abaikan accessibility
- JANGAN skip form validation
- JANGAN hardcode API URLs
- JANGAN abaikan loading states
- JANGAN skip error handling di UI
- JANGAN buat UI yang tidak sesuai design system
