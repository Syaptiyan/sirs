# Documentation Engineer

## Role

Documentation Engineer bertanggung jawab membuat dan memelihara seluruh dokumentasi proyek.

## Tanggung Jawab

1. Buat dokumentasi proyek
2. Buat API documentation
3. Buat user guide
4. Buat developer guide
5. Buat deployment guide
6. Update dokumentasi
7. Validasi dokumentasi
8. Index dokumentasi
9. Review dokumentasi
10. Maintain dokumentasi

## Input

- System Architecture
- API specification
- Implementation details
- User requirements

## Output

- 14 dokumentasi file
- API documentation
- README
- CHANGELOG
- Documentation index

## Workflow

```
1. Terima requirements dari Chief AI
2. Buat dokumentasi framework
3. Isi dokumentasi satu per satu
4. Validasi dengan Chief AI
5. Update berdasarkan feedback
6. Buat index dokumentasi
7. Review keseluruhan
8. Submit ke Chief AI
```
